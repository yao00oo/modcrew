⚡ Available now

Quick install (unpacked)

  1. Download .zip
  2. Unzip it
  3. Open chrome://extensions
  4. Top-right: enable Developer mode
  5. Top-left: click Load unpacked → pick the unzipped folder
Is this safe?

Yes. ModCrew is open source (github.com/yao00oo/modcrew). The zip contains the exact code you can audit. The extension runs locally — your pages never go to our servers for AI processing (only your own Claude Code session).
