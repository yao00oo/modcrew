# modcrew-mcp

CLI helper for [ModCrew](https://modcrew.dev). The actual MCP relay runs in the cloud at `api.modcrew.dev` — you don't need to run anything locally.

## What this does

```bash
npx modcrew-mcp
```

Opens `modcrew.dev/install` in your browser. There you'll find a one-line `claude mcp add` command to copy.

## Why this package exists

So you can discover ModCrew via:
- `npx modcrew-mcp` (quick install opener)
- npm search "modcrew"

See full docs at [modcrew.dev](https://modcrew.dev).
